
# Requirements
* Node 8
* Git


# Installation
* Clone project repository
> ``` git clone https://github.com/Jycer/LNR-Node-Express.git```

* Install the dependencies in the local
> ```npm install```


# Staring the service
After running the following command, the LNR service will run on http://127.0.0.1:5000/
> ```npm start```
